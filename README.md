# AI Interaction Tracking Web App

## Overview
A web application for precise AI cost tracking and interaction monitoring.

## Features
- Track AI model usage
- Monitor API calls and costs
- Detailed interaction analytics

## Tech Stack
- React
- Tailwind CSS
- Netlify Deployment

## Setup
1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm start`
